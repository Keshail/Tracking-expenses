## Описание
Приложение для учёта личных расходов с возможностью отслеживания по категориям, установки месячных лимитов и формирования отчётов.

## Возможности
- Добавление, просмотр, удаление расходов.
- Управление категориями (добавление, изменение лимита, удаление).
- Фильтр расходов за текущий месяц или произвольный месяц.
- Отчёты: сумма за месяц, детализация по категориям, превышение лимитов, топ-3 затратных категории.
- Хранение данных в SQLite (файл `expenses.db`).

## Требования
- Windows 7/10/11 (для `.exe`)
- Или любой компьютер с Python 3.10+ (для запуска исходного кода)

## Запуск

### Через исполняемый файл (рекомендуется)
1. Скачайте папку з апроектом.
2. Запустите `ExpenseTracker.exe`.
3. При первом запуске автоматически создастся база данных `expenses.db` в той же папке.

### Из исходного кода (Python)
1. Убедитесь, что установлен Python 3.10+.
2. Откройте терминал в папке проекта.
3. Выполните: `python expense_tracker.py`

## Использование
- Вкладка «Расходы» – добавление/удаление расходов, фильтр по месяцу.
- Вкладка «Категории» – добавление категорий, установка месячных лимитов.
- Вкладка «Отчёты» – выбор месяца, просмотр общей суммы, расхода по категориям, превышения лимитов и топ-3 категорий.

## Структура базы данных
- `categories` – (id, name, monthly_limit)
- `expenses` – (id, category_id, amount, date, note)
